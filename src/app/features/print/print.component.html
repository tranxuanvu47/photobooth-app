Print Options

Print Method

Browser Print

Use your browser's print dialog. Works on all devices.

Direct Print (Agent)

Print directly to a system printer without dialog.

Print Agent Not Running

For direct printing without dialogs, start the print agent on your PC.

How to start the print agent
  1. Open a terminal/command prompt
  2. Navigate to the project folder
  3. Run: npm run print-agent
  4. The agent will start on port 3000
